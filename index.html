This site is a place to store the projects I’ve made as an Applied Media Communications student. I want to be able to come back in the future and compare where I started to where I end up.

If you’re in media too, I hope this makes you feel reassured: nobody’s first work is perfect. Things improve a lot as you practice, mess up, and keep going.
